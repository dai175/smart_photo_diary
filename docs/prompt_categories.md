# 感情深掘り型プロンプトカテゴリ設計仕様

感情中心のライティングプロンプトシステムの設計仕様書。
すべてのプロンプトは感情探求と内面深化を目的としている。

> **注**: UI表示用のカテゴリ情報は `lib/utils/prompt_category_utils.dart` を参照。

---

## カテゴリ一覧

| カテゴリ | enum値 | 対象プラン | Basic数 | Premium数 |
|----------|--------|-----------|---------|-----------|
| 感情 | `emotion` | Basic + Premium | 5 | 0 |
| 感情深掘り | `emotionDepth` | Premium | 0 | 2 |
| 感情五感 | `sensoryEmotion` | Premium | 0 | 2 |
| 感情成長 | `emotionGrowth` | Premium | 0 | 2 |

| 感情幻想 | `emotionFantasy` | Premium | 0 | 1 |
| 感情癒し | `emotionHealing` | Premium | 0 | 2 |
| 感情エネルギー | `emotionEnergy` | Premium | 0 | 2 |

## 各カテゴリ詳細

### 感情（Emotion）

- **対象**: Basic・Premium共通
- **説明**: 写真から感じる基本的な感情を探求し、自己理解を深める

**設計方針**:
- 写真から感じる素直な感情に焦点を当てる
- シンプルで親しみやすい感情表現を促進
- 感情の言語化を支援する
- 自分らしさや心の動きを大切にする
- 感謝や温かさなどポジティブ感情も含む

**期待される効果**:
- 感情の言語化スキル向上
- 自己理解の深化
- ポジティブな感情体験の認識
- 写真と感情の結びつき強化
- 日記習慣の継続促進

**Basic用テーマ**: 最初に感じた気持ち / 心が動いた理由 / 温かい感情 / 感謝の気持ち / 自分らしさ

**推奨タグ**: 感情, 気持ち, 心, 温かい, 感謝, ありがたい, 自分らしさ, 素直, 第一印象, 直感

---

### 感情深掘り（Emotion Depth）

- **対象**: Premium限定
- **説明**: 感情の背景や変化プロセスを詳細に探求し、深い自己洞察を促進する

**設計方針**:
- 感情の根本的原因の探求
- 時間的変化の追跡
- 深い内省の促進
- 感情パターンの認識
- 自己理解の深化

**期待される効果**:
- 感情的自己認識の向上
- 感情調整能力の発達
- 深い内省習慣の形成
- 感情パターンの理解
- 心理的成熟の促進

**Premiumテーマ**: 感情の背景・キッカケ / 感情の時間的変化

**推奨タグ**: 感情深掘り, 背景, キッカケ, 変化, 時間, 原因, 進化, 深層, 探求, 洞察

---

### 感情五感（Sensory Emotion）

- **対象**: Premium限定

**設計方針**:
- 五感と感情の結びつき探求
- 感覚記憶の活用
- 多感覚的体験の記録
- 感情の立体的理解
- 記憶の鮮明な保存

**期待される効果**:
- 感覚と感情の統合理解
- 記憶の鮮明度向上
- 体験の多層的記録
- 感情体験の豊かさ向上
- 感覚的感受性の発達

**Premiumテーマ**: 音・においの感情喚起 / 場所の空気感・心への影響

**推奨タグ**: 感情五感, 音, におい, 空気感, 雰囲気, 感覚, 記憶, 喚起, 影響, 体験
